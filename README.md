# docker-desktop-intellij-ide

## Description
This is a mobil developer evironment.
IDE preferences: ```prefs```
Projects: ```workspace```
Maven local repository: ```maven```

## Tech stack
- intellij

## Docker stack
- ubuntu

## To run
```sudo ./install.sh -u```

## To stop
```sudo ./install.sh -d```

## To see help
`sudo ./install.sh -h`
